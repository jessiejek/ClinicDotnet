## 2. Root Cause

### File
`src/app/portals/public/components/step-payment/step-payment.component.ts`

### Function
`submitBookingRequest(payload: CreateBookingRequest)`

### Old (broken) behavior

The method intentionally **discarded** the booking payload and sent an empty body to the API:

```typescript
private async submitBookingRequest(payload: CreateBookingRequest) {
  void payload;  // ← PAYLOAD DISCARDED
  const createdRow = await firstValueFrom(this.apiService.post<any>('bookings', {}));
  //                                          empty body sent → ^^
  const bookingId = trimOptionalString(createdRow?.booking_id) ?? ...
  //                                       ↑ Supabase snake_case field name
  if (!bookingId) {
    throw new Error('API create_booking did not return a booking id.');
    //       ↑ old Supabase RPC reference in error message
  }
  ...
}
```

### Old API call
`apiService.post<any>('bookings', {})` — empty `{}` body

### Old payload
Discarded via `void payload` — the `CreateBookingRequest` was never sent.

### Old response parsing
- Checked `response.booking_id` (Supabase snake_case convention)
- Error message referenced `create_booking` (Supabase RPC name)

### Why it failed
The `POST /api/bookings` endpoint received `{}` with no `doctorId`, `serviceIds`, `appointmentDate`, etc. The backend validator rejected it or (in the earlier Supabase flow) the call failed because it was trying to call a Supabase RPC that didn't exist or had auth issues. The browser showed:  
**`"Failed to create booking in Supabase."`**


### After (fixed)
```typescript
private async submitBookingRequest(payload: CreateBookingRequest)
    : Promise<{ id: string; queueNumber: number | null }> {
  const body: Record<string, unknown> = {
    doctorId: payload.doctorId,
    serviceIds: payload.serviceIds ?? (payload.serviceId ? [payload.serviceId] : []),
    appointmentDate: payload.appointmentDate,
    slotStartTime: payload.slotStartTime,
    slotEndTime: payload.slotEndTime
  };
  if (payload.notes) {
    body['notes'] = payload.notes;
  }
  const booking = await firstValueFrom(this.apiService.post<any>('bookings', body));
  const bookingId = booking?.id;
  if (!bookingId) {
    throw new Error('Booking was created but no booking ID was returned.');
  }
  return {
    id: bookingId,
    queueNumber: booking?.queueNumber ?? null
  };
}
```
